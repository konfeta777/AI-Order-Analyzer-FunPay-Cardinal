# AI Order Analyzer - Плагин для анализа заказов

## 🎯 Описание

Плагин собирает оплаченные заказы за выбранный период, выгружает переписки и формирует ZIP-архив для анализа в ChatGPT. Автоматически классифицирует заказы на категории:

- ✅ **ЗАБЫЛ ПОДТВЕРДИТЬ** - клиент получил товар, всё хорошо, но забыл нажать кнопку подтверждения
- ⚠️ **СПОРНАЯ СИТУАЦИЯ** - есть претензии, недопонимание, возможный конфликт  
- ❓ **НЕЯСНО** - недостаточно информации для определения

## 💡 Преимущества

✔ Экономия часов ручного перечитывания переписок  
✔ Быстрое выявление проблемных ситуаций  
✔ Понимание, где теряются подтверждения  
✔ Удобная подготовка к обращениям в поддержку  
✔ Автоматическая классификация через ChatGPT

## ⚙️ Настройки

### Основные параметры:

- **Период анализа** - количество дней для сбора заказов (по умолчанию: 7)
- **Минимальная сумма заказа** - фильтр по цене в рублях (по умолчанию: 0)
- **Количество сообщений** - максимум сообщений из каждого чата (по умолчанию: 50)
- **Промпт для ChatGPT** - можно настроить свой промпт для анализа

### API ключ:

Плагин использует OpenRouter API с бесплатной моделью `arcee-ai/trinity-large-preview:free`.  
По умолчанию используется общий ключ, но вы можете указать свой в конфигурации.

## 📱 Telegram-команды

### `/analyze_orders`
Запускает процесс анализа заказов:
1. Собирает оплаченные заказы за указанный период
2. Выгружает переписки по каждому заказу
3. Создает ZIP-архив с текстовыми файлами
4. Отправляет данные в ChatGPT для анализа
5. Возвращает классификацию и рекомендации

### `/show_settings` или `/ai_order_settings`
Открывает меню настроек плагина с возможностью:
- Включить/выключить плагин
- Изменить период анализа
- Настроить минимальную сумму заказа
- Изменить количество сообщений
- Редактировать промпт для AI
- Сбросить промпт к значению по умолчанию

## 🚀 Использование

1. Включите плагин через `/show_settings`
2. Настройте параметры анализа (период, сумма, количество сообщений)
3. Запустите анализ командой `/analyze_orders`
4. Дождитесь результатов:
   - ZIP-архив с переписками
   - Автоматический анализ от ChatGPT с классификацией

## 📊 Формат выгрузки

Каждый заказ сохраняется в отдельный текстовый файл со структурой:

```
============================================================
ЗАКАЗ #12345
============================================================
Покупатель: username
Товар: Название товара
Сумма: 500 RUB
Дата: 2024-01-15

ПЕРЕПИСКА:
------------------------------------------------------------
[Покупатель]: Здравствуйте, хочу купить
[Продавец]: Добрый день! Оформляйте заказ
[Покупатель]: Спасибо, всё получил!
```

Идеально подходит для:

- Тех, кто хочет быстро выявлять проблемные ситуации
- Подготовки аргументов для обращения в поддержку FunPay

## 🔧 Технические детали

- **Модель AI**: Arcee Trinity Large (OpenRouter)
- **Формат архива**: ZIP с UTF-8 кодировкой
- **Максимальный размер анализа**: ~50,000 символов
- **Таймаут запроса**: 60 секунд
- **Многопоточность**: Анализ выполняется в отдельном потоке

## ⚠️ Ограничения

- Анализируются только оплаченные заказы (статус PAID)
- Максимум 50,000 символов для одного запроса к AI
- Один анализ за раз на пользователя
- Требуется активное подключение к интернету

## 📝 Примечания

- Все данные обрабатываются локально, в ChatGPT отправляется только текст переписок
- Архив можно сохранить для дальнейшего анализа или отправки в поддержку
